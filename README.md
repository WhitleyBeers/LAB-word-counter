# LAB: Word Counter

In this lab, you will be using your HTML, CSS and some basic JS skills to create and style a word counter.

## Get Started
- Navigate to your `~/workspace/foundations/inclass` directory

```bash
$ git clone {YOUR REPO LINK}
$ cd LAB-word-counter
```
